如何保存微信小视频

今天朋友说微信小视频传到微信网页版上，可以看，但是点右键下载以后是大小为0的MP4。问怎么破？

我看了下，用chrome的话，至少2个办法。

办法一，用Chrome的Copy as cURL

  1. 点击右上角“三”按钮->更多工具->开发者工具，在下面的开发者工具栏选择Network，再下面2行选Media 这个样子
  2. 用文件传输助手把视频转发给自己，然后下面是这个样子
    下面多了2行，其中一行Response的Content-Length:0 也就是大小为0。点击另一行，Content-Length:291195 有内容了。
  3. 右键点击有内容的，选择"Copy as cURL",会得到一条curl指令
  4. 如果你电脑上安装了curl，那么打开命令行，粘贴这条指令，在指令末尾加上 > abc.mp4 ,稍等一下，视频abc.mp4就下载完成了。

办法二，从Chrome的缓存里找

  1. 找到缓存位置打开

    Windows Vista: C:\Users[USERNAME]\AppData\Local\Google\Chrome\ Windows XP: C:\Documents and Settings[USERNAME]\Local Settings\Application Data\Google\Chrome\ Windows 7/8: C:\Users[USERNAME]\AppData\Local\Google\Chrome\User Data\Default\Cache Mac OS X: /Users/[USERNAME]/Library/Caches/Google/Chrome/

  2. 根据文件创建时间，找到疑似文件。把扩展名改为.mp4 试试看。能放就OK了。
